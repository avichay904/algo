# ברוכה הבאה, מעיין! 🌸

הכלי הזה הוכן במיוחד בשבילך, אהובה שלי, לקראת המבחן באלגוריתמים 1.

כל האלגוריתמים מסיכום הקורס מחכים לך בתפריט שבצד ימין. לכל אלגוריתם יש **הדמיה חיה** — רואים איך הצבע נמצא, איך הטבלה מתמלאת, איך הגרף נסרק — ו**הסבר בעברית**.

---

## איך להשתמש?

1. **בחרי קטגוריה** מהתפריט בצד ימין (למשל "תכנון דינמי")
2. **בחרי אלגוריתם** מתוכה (למשל LCS)
3. מופיעות **3 לשוניות**:
   - **הסבר** — תיאור של הבעיה והאלגוריתם
   - **קוד** — הקוד עצמו (בשפות שונות)
   - **הדמיה** — אזור ההדמיה
4. לחצי **הפעל** (הכפתור הירוק למעלה) כדי להתחיל את ההדמיה
5. השתמשי בכפתורי **הבא / הקודם** כדי לצעוד צעד-צעד
6. שני את **המהירות** אם זה מהיר מדי

---

## מה יש בתפריט?

### 📐 קטגוריות עם הדמיות מוכנות
- **תכנון דינמי** — Knapsack, LCS, LIS, סדרת פיבונאצ'י, מרחק עריכה, KMP
- **מיון וברירה** — Quicksort / Quickselect, Merge Sort
- **חיפוש בגרפים** — BFS, DFS, בדיקת גרף דו-צדדי, Rabin-Karp
- **אלגוריתמים חמדניים + MST** — קרוסקל, פרים, דייקסטרה, תזמון משימות

### 📚 נושאים תיאורטיים (14 הערות עברית)
הנושאים שאין להם הדמיה חיה — אבל יש להם דף הסבר בעברית עם פסאודו-קוד, סיבוכיות וטיפ למבחן:
- רקורסיה ומשפט האב
- חציון החציונים (Quickselect דטרמיניסטי)
- מציאת שיא (Find-Peak)
- בעיית הקפיצה (Jump DP)
- חיתוך מוט / ספירת מסלולי לוח
- קבוצה בלתי תלויה מרבית בעץ (MIS on Trees)
- בדיקת גרף דו-צדדי — הסבר תיאורטי
- סיווג קשתות ב-DFS (Tree/Back/Forward/Cross)
- החלפת מטבעות חמדני
- עץ סיומות (Suffix Trie / Tree)
- התאמה מעגלית / DNA בקירוב
- כפל שרשרת מטריצות
- קידוד האפמן
- בחירת פעילויות (חמדני)

## טיפים אחרונים

- השתמשי בחיפוש בראש התפריט אם את יודעת שם של אלגוריתם
- אם יש נוסחת נסיגה מסובכת — קראי קודם את ההסבר, ואז הפעילי את ההדמיה כדי לראות אותה חיה
- עם כל הדמיה שימי לב איך היא "בונה" את הפתרון, תהיה לך הבנה עמוקה יותר ללא שינון
