# מעיין לומדת אלגוריתמים 🌸

ברוכה הבאה, מעיין! 💕

הפרויקט הזה הוכן במיוחד בשבילך לקראת המבחן באלגוריתמים 1.

## הפעלה (פעם ראשונה)

1. פתחי טרמינל ב-VS Code / PowerShell בתיקיית הפרויקט
2. הריצי:
   ```
   npm install --legacy-peer-deps
   npm start
   ```
3. הדפדפן ייפתח אוטומטית על `http://localhost:3000`

## הפעלה (פעמים הבאות)

```
npm start
```

## אם משהו נתקע

- `Ctrl+C` בטרמינל להפסיק
- `npm start` להפעיל שוב
- אם עדיין לא עובד: מחקי את `node_modules` והריצי `npm install --legacy-peer-deps` מחדש

## מה יש פה?

- **4 קטגוריות של אלגוריתמים** עם הדמיות אינטראקטיביות (ממוינות לפי סיכום הקורס)
- **14 הערות עברית** לנושאים תיאורטיים (רקורסיה, חציון החציונים, קידוד האפמן וכו')
- עיצוב ורוד-קרם-סגלגל לחמימות 🌸

בהצלחה במבחן, אהובה! 💕
